# The Workbench

Static Astro portfolio/project hub for The Workbench.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

- /
- /readyroll/
- /agents/
- /about/

## Deployment

GitHub Pages deployment is configured via `.github/workflows/deploy.yml`.
Push to `main` to trigger build + deploy.

## Notes

- External links are centralized in `src/data/links.ts`.
- Design reference screenshots in `/design-reference` are not included in public output.
